Thoughts on Progressive Enhancement

I think progressive enhancement is the best strategy in web design because it emphasizes the web content before anything else. It's letting have access to the content to the web developer to give their client an easy ride to change whenever they want without touching other features such as CSS or JavaScript. This makes the web page load faster than other strategies. Suppressive enhancement is an evolution for all the web design strategies known as Graceful Degradation.
